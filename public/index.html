Rules
  1. Please donot copy code from some other site and paste it here.
  2. Please do not share sensitive information or Objectional content
  3. Please do not try to brick the styling of the page. In doing so you are degrading the experiance of people actually in need. Keep things fair
  4. Please include the ID specified in the Add code box to the parent div of your code.
  5. If any of the code is found not working or hindering with the other Loaders, it will be removed
  6. If You want to contribute, write a proper working code.
  7. You can see the output of the code in the output box.
  8. In the styling of your parent div give a height of 300px X 300px or less. By default it is set at 300px X 300px with a solid black border of 1px and a white background
About

This is a site which provide CSS loaders for your porject so you can focus on getting it to work. This site is open to all. Anyone can contribute to site by adding a code. You can also check the source code of the application on github.com.
This site is hosted on Digital Ocean and MySQL is used as DBMS. The icons have been provided by Icons8. The backend is written by node and express. I am not using any CSS library. Though Jquery is used for javascript and making calls to the server.
